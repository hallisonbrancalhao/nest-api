# **Steps**

### Import Mongoose App module

https://docs.nestjs.com/techniques/mongodb#plugins

### Generate module

```
nest g module auth
```

### Gerando controller

```
nest g controller auth
```

### Gerando Service

```
nest g service auth
```

### Instalando bcrypt

https://docs.nestjs.com/security/encryption-and-hashing#hashing

```
npm i bcrypt
npm i -D @types/bcrypt
```
